Test Angular

L'objectif de cette excercice est de vous accompagner pour avoir une appréciation de votre maîtrise du Framework Angular, de typescript/javascript, CSS/SCSS et votre capacité a assimiler les besoins fonctionelles...

Veuillez lancer l'API de test npm run start:api
Vous devriez voir le message "Hello .arianee" sur http://localhost:3000

Vous pouvez piocher de la liste des tâches a votre convenance, pas d'ordre et aucune obligation a tout finir, bien sûr vous êtes autorisé a utiliser internet pour rechercher n'importe quelle informations : )
